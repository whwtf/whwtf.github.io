部署 Code-Server，随时随地 VsCode

2021-06-05 Programing Code-Server 评论

Code-Server 是一个可以在远程服务器上运行 VS Code 的工具。拥有与 VS Code 完全相同的界面和使用方法。

将其部署在远程服务器上，就可以通过 Edge、Chrome 等浏览器在任何设备上运用完全一致的环境进行写作与开发，一切计算都在服务器上进行，不用考虑终端设备的性能。

在Releases页面找到相应的 rpm 或者 deb 软件包下载并上传到远程服务器（也可以直接在服务器上使用 wget 等工具直接下载）并安装。
